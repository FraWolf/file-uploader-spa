# File Uploader SPA

### Frontend

- Vite
- Vanilla JS
- Boostrap 5.1

#### Start Frontend

```
yarn dev
```

or

```
npm run dev
```

### Backend

- Node.js
- Typescript
- Mongodb
- Mongoose
- Express

#### Start Backend

Fill the .env file inside backend/ folder

```
yarn dev
```

or

```
npm run dev
```
